Place your release APK here as:

  NexaERP.apk

After building the Android app (e.g. from apps/android-app), copy the generated APK
to this folder, commit, and redeploy the frontend. The login page "Download APK"
button uses: https://YOUR-SITE/downloads/NexaERP.apk

Alternatively set REACT_APP_ANDROID_APK_URL in hosting env to any full URL
(S3, GitHub Releases, Drive direct link, etc.).
